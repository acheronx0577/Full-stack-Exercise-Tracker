## Installation

1. Clone the repository:
   ```
   git clone git@github.com:acheronx0577/Full-Stack-Exercise-Tracker.git
   ```
   ```
   cd Full-Stack-Exercise-Tracker
   ```
2. Go to extension vs code and install MongoDB for VS Code
3. After that follow its intruction to connect to mongodb
5. Install dependencies:
   ```
   npm install mongoose
   ```
   ```
   npm init
   ```
   ```
   npm init -y
   ```
6. Start in http://localhost:3000
   ```
   npm start
   ```
